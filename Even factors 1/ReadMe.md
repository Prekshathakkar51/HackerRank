## Given a number N, print the even factors of N.If the even factor does not exists for N print '-1'.

Input Format

A number `'N'`

Constraints

- 1 <= N <= 1000

Output Format

All the integers which are even factors of `'N'`. If there are no even factors print `-1`

````
Sample Input 0

8
Sample Output 0

2 4 8
````

````
Sample Input 1

9
Sample Output 1

-1
````
